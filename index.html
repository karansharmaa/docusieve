AI-Powered Resume Analyzer (Local LLM)

DocuSieve is a backend project built with FastAPI and a Groq API (via Ollama). It compares a PDF resume against a job description and generates a match score, keyword overlap stats, strengths, weaknesses, and improved bullets.

This Vercel page is just a landing page for now. The actual API runs locally on: http://127.0.0.1:8000/docs.

Current capabilities

  • PDF resume text extraction
  • Keyword overlap and match scoring
  • Local LLM feedback: strengths, weaknesses, improved bullets

Planned

  • Hosted backend with online LLM (OpenAI/Anthropic)
  • Full web UI to upload resumes and job descriptions
  • Semantic similarity scoring and ATS-style reports
